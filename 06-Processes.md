# Processes and Process Management

## What it is
A process is an instance of a running program. Linux provides tools to monitor and control processes.

## Common commands practiced
```bash
ps
top
htop
kill

## Process states
- Running
- Sleeping
- Stopped
- Zombie

## Why it matters in cybersecurity
Analyzing running processes helps detect malicious programs, resource abuse, and unauthorized activity.

## Key takeaway
Understanding processes is essential for system monitoring and incident response.
