# Windows Networking Basics

## What it is

Windows provides built-in networking tools to inspect IP configuration, active connections, and network status.

Understanding these tools is important for troubleshooting and security analysis.

## Key Networking Concepts

- IP Address
- Subnet Mask
- Default Gateway
- DNS Server
- MAC Address

## Common Commands Practiced

```powershell
ipconfig
ipconfig /all
netstat -ano
ping
tracert

## Why it matters in cybersecurity

Attackers rely on network communication to move laterally, exfiltrate data, or maintain persistence.  
Security professionals use networking commands to identify open ports, active connections, and suspicious traffic.

## Key takeaway

Basic Windows networking knowledge is essential for system enumeration and security monitoring.
